Photostash

Overview
PhotoStash is a photosharing application that allows users to create albums, invite their friends and share photos with them in shared albums. Users create accounts, then create albums, invite their friends to join albums and if the request is accepted both parties can see the photos in the album. Users can create autoupload albums which means that all photos taken between two timestamps will automatically be uploaded to the album contents from their iPhone photos library. Users can toggle autoupload to their preferred setting.

Technology
PhotoStash was built using Swift to create the front end and firebase to handle the storing of user, album and photo information. Built in a team of three for the class software engineering workshop at WashU. In this application I was responsible for linking up the firebase databases to the front end of the application and fill in the UI with user data.

Seattle Low Income Housing Map

Overview
Interactive map that shows the amount of low income housing units per zip code in the Seattle area. This project also shows how the number of low income housing units added to the housing market in Seattle has progressed over the years. Was created to accompany a Seattle Times article regarding political emphases on low income housing in the mayors office.

Technology
This product was built using d3, JavaScript and the Leaflet maps library. I constructed this visualization for my Information Visualization class at WashU.

Systems Biology Internship

Overview
Undergraduate research internship at the Institute for Systems Biology in Seattle. During this internship I worked on a metabolic model of a methanogen, an organism that produces methane. My team was attempting to reverse engineer metabolic pathways in Methanoccocus Maripalus to have it produce useful byproducts. Worked in Python and MATLAB on this project. Implemented a tool, gapfilling that filled in missing reactions in the metabolic model.

Technology
Worked on model in both Python and MATLAB. For this project I learned to use to CobraPy library in order to implement the gapfilling tools. Also gained experience working with the Python pandas library to manage large datasets of reactions and metabolic pathways.

UMates

Overview
UMates is application designed to help Washington University students find prospective roomates. Users sign up and set various roomate preferences such as gender, cleanliness and noise level. They are then shown a list of prospective roomates that they can invite to connect, at which point they will recieve that users email so they can contact them directly.

Technology
UMates was built using Swift to build the front end and Firebase to construct the back end. Wireframing and high fidelity mock ups were made using Sketch. I was responsible for constructing the front end of this application in Swift and providing assistance to the back end team who linked up Firebase to the app.
